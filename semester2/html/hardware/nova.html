Data General Nova

NOVA

Image courtesy of Computer History Museum

"The best small computer in the world". That was the heading of the brochure for the NOVA.1 Data General started producing the NOVA in 1969. It was one of the first 16-bit computer and led the way toward word lengths that were multiples of the 8-bit byte. The other really important aspect is, that it was first to employ medium-scale integration (MSI) circuits.2 The base price was $3,995, but normally RAM was added in the form of core memory, which brought the price up to $7,995. By 1970, 700 units were shipped.
Even high-schooler Stephen Wozniak, Apple's future co-founder, was said to be enchanted with the Nova's elegantly designed architecture

SuperNOVA

Supernova had the same basic organization as the NOVA: 16-bit word, 4 accumulators, flexible I/O facility, interchangeable core and read-only memories and the same packaging design.

Faster

The SuperNOVA used ROM for library software. A full memory cycle using core memory took 800 nanoseconds for the Supernova. Using read-only memory, the cycle time was 300 nanoseconds. The NOVA took 400 nanoseconds longer for a full memory cycle, 1200 nanoseconds compared to the 800 nanoseconds.1

Later versions

Nova 840

In 1970/71 the Nova 800 and Nova 1200 series replaced the NOVA and SuperNOVA. The next version was the Nova 2, which was shipped in 1973 and the Nova 3 in 1975. Originally the Nova 3 was supposed to be the last Nova machine and Data General wanted to push the machine "Data General Eclipse", but because of high demand a Nova 4 computer was shipped. In total 50'000 Nova computers were sold.

This machines influenced the design of the Xerox Alto and the Apple I. Furthermore, its architecture was the basis for the Computer vision CGP.2
