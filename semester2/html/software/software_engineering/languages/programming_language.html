What is a programming language?

A programming language is a set of grammatical and semantic rules used to communicate with a computer or a machine. A programming language is used to create programs, which are a set of instructions that control the behavior of a computer. They can also be used to express algorithms1, which is basically a set of finite instructions, which solve a particular problem.

Unlike natural languages, such as English, programming languages cannot be ambiguous because computers are not able to make choices in case of ambiguity.

There are many programming languages, which are applied differently to create software. Thus, programming languages are also used to solve problems, because the process of creating software2 is mainly the process of analyzing and solving a problem.

A programming paradigm is a fundamental style of programming, or way of building and creating the structure of a computer program. There exist many programming paradigms, such as structured programming, functional programming, imperative programming, and object-oriented programming. Consequently, the study of these paradigms is very important in software engineering3.

When we talk about programming languages, we usually refer to high-level programming languages, such as C, and not the language that computers really understand, which is a sequence of 1s and 0s. Programs written with high-level programming languages first have to be translated to the machine language in order to be executed by the CPU4. There are essentially 2 types of translation: compilation5 and interpretation6. Compilation is usually done through the use of a program called a compiler, which usually directly translates the source code7 to the machine language8. The main actor in interpretation is the interpreter, which reads and executes the program instruction by instruction.

Languages that are similar to the way a computer works are called low-level programming languages9. Languages that are similar the way the humans think are called high-level programming languages10.
