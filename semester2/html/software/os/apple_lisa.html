APPLE LISA

The Lisa hardware, operating system, and its set of office applications were designed by Apple to work together to create a powerful document processing workstation.

The Lisa operating system featured preemptive multitasking and protected memory - extremely advanced features for a personal computer at the time. Partly based on advanced elements from the Apple III SOS operating system released three years earlier, the Lisa also organised its files in hierarchal directories, making the use of large hard drives practical. The Macintosh eventually adopted this disk organisational design as well for its HFS filing system.

Conceptually, the Lisa resembles the Xerox Alto in the sense that it was envisioned as an office computing system. Lisa has two main user modes - the Lisa Office System and the Workshop.

The Lisa Office System, released in 1983 by Apple Computers with their Lisa microcomputer, is the GUI environment for end users. It was eventually renamed "7/7" in reference to the seven supplied application programs: LisaWrite, LisaCalc, LisaDraw, LisaGraph, LisaProject, LisaList, and LisaTerminal. The Workshop was a program development environment and was almost entirely text-based, though it used a GUI text editor.

Notably, developed for the Apple Lisa computer, the Lisa project was the first graphical project management software. It was conceived and implemented by Debra Willrett of SoloSoft.
