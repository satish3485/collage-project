Graphical User Interface

Xerox Alto

the Xerox Alto

Completed in 1973, one of Xerox Palo Alto Research Center's (PARC) first products was the Xerox Alto. It was the first computer to have a mouse and a graphical user interface (GUI), inspired by Engelbart's oN-Line System.

The display had the same size and orientation of a normal sheet of paper, and used a grid of bitmapped pixels for the display. The graphical component only had boxes and text as elements and it used Bravo, a software developed to show different kinds of fonts and text sizes on the display.

The Alto was never sold to the public. However, Xerox started selling the Xerox Star in 1981, which had a lot in common with the Alto.

Apple Lisa and Macintosh

the Lisa Office System

Inspired by the work of PARC engineers, Apple decided to create its own GUI. In 1983, the Apple Lisa was brought to the market. It had a lot in common with the Xerox Alto, but it implemented other functions, like icons, which were missing in the Xerox, a single button mouse which utilised a double click to open applications, and the click and drag.

In 1984, the Macintosh successfully hit the market. It wasn't the first computer to have a graphical user interface, but it was much cheaper than its competitors. The arrival of Aldus PageMaker for the Macintosh in 1986 and the color displays of the Macintosh II amplified the success of Apple's products.


Graphical Environment Manager

GEM version 1.1

Created by Digital Research Inc. (DRI), the GEM (Graphical Environmental Manager) was the successor of the GSX, a graphic library whose purpose was to allow graphic applications to run on different machines that supported it. It was supposed to be a program that enabled a complete graphical user interface on personal computers.

The first version of the GEM, GEM desktop 1.0, came to the market in 1985. It was compatible with CGA and EGA displays, and for some machines an 8-bit color version was also available. It was only supposed to work on IBM machines, but due to numerous complaints, this limitation was soon removed. Not much time later, Apple sued DRI because of the strong similarities between GEM and the Apple products (namely Lisa and Macintosh).

As an answer to Apple, in 1986 the second version was released, called GEM desktop 2.0. This version had graphical differences compared to the older version, meant to solve similarity issues with its competitors. This version also had other improvements, like bug fixes and the addition of VGA to the list of compatible displays. The last commercial version of GEM, 3.0, came out in 1988, bringing some pre-included applications and a general performance upgrade.
