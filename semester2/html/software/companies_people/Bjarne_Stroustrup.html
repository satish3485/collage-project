Bjarne Stroustrup

Bjarne Stroustrup

Bjarne Stroustrup1, born on December 30, 1950, is a Danish computer scientist best known for the creation and development of the C++ (Cee plus plus)2 programming language . Stroustrup received his master's degree from Aarhus university in Denmark, and then went on to receive a PhD in computer science at the University of Cambridge, England 3, in 1979.

C++ Development

He started developing C++ in 1978, then called C with classes. Bjarne's goal was to add object oriented programming into the C language. His version of the language included classes, default function arguments, inlining, basic inheritance and strong type-checking, in addition to all the features provided by C.

In 1983, C with classes was renamed C++. Many features had been added to the language by this time, notably fucntion overloading, virtual functions, references with the “&” symbol and single line comments using the two forward slashes (taken from the language BCPL4).

In 1985, Bjarne's reference to the language, entitled The C++ Programming Language5 was published: it was a very important reference as the language had not been officially standardized yet. In the same year, C++ was implemented as a commercial product.
In 1990, The Annotated C++ Reference Manual6, written by Margaret Awas, was released.

Stroustrup has also won several awards, and he is highly respected in his field. He holds the College of Engineering chair in computer science at the Texas AM University7 and is a visiting professor at the Columbia University.8
